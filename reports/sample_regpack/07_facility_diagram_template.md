# Facility Diagram — Template

**Facility:** Crowe Logic Psychedelic Research Laboratory
**Address:** 3201 W Thomas Rd, Suite E, Phoenix, AZ 85017

---

## Purpose

DEA applications for Schedule I handling benefit from a facility diagram
that shows the storage location relative to entrances, exits, adjacent
occupancy, and the alarm/monitoring layout. This template is a prompt
for the diagram — the actual diagram must be produced by the applicant
(typically as a floor plan PDF attached to the application).

## Required elements

The diagram should clearly show:

1. **Building exterior** — entrances, exits, adjacent occupancy
2. **Facility footprint within the building** — which rooms belong to
   the applicant
3. **Schedule I storage location** — marked prominently
4. **Perimeter security elements** — door sensors, window sensors,
   motion detectors
5. **Alarm control panel** location
6. **Emergency exits** — marked but distinct from routine-use doors
7. **Surveillance camera coverage** (if installed) — camera positions
   and coverage zones
8. **Adjacent occupancy** — what is on the other side of each wall
   adjoining the storage area (especially if the wall is shared with a
   public corridor or uncontrolled space)

## Schedule I storage area — narrative

> Dedicated 120 sq ft laboratory room with reinforced door (solid-core, deadbolt lock with restricted key control). Schedule I substances will be stored in a GSA Class 5 security container affixed to the building structural wall. The container is located against an interior wall not shared with any public corridor. Motion detectors cover the storage area and perimeter door; central station alarm monitoring is in place 24/7 via a UL 2050 Protection Level II certified provider.


## Diagram checklist

- [ ] Scale or "not to scale" annotation
- [ ] North arrow
- [ ] Date of preparation
- [ ] Preparer's name and credentials
- [ ] Key / legend for security symbols
- [ ] Schedule I storage safe location marked
- [ ] Alarm sensors and zones indicated
- [ ] Entry/exit doors distinguished from internal doors
- [ ] Adjacent space usage labeled (office, lab, corridor, etc.)
- [ ] Signatures of PI and facility manager (if different)

## Suggested tools

- Microsoft Visio
- Lucidchart
- OmniGraffle
- Adobe Illustrator
- AutoCAD (if a precise measured drawing exists from building records)

Attach the finalized diagram as a PDF to the DEA application packet.
